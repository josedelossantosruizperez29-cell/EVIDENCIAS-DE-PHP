Tienda de Libros

Construye una aplicación PHP usando el patrón Model · View · Controller

60 – 90 min Individual PHP 8+ 100 puntos

¿Qué vas a construir?

Una Tienda de Libros con dos páginas PHP organizadas bajo el patrón MVC.

El reto no es que quede perfectamente estilizada, sino que el código esté correctamente repartido entre Model, View y Controller, siguiendo las reglas del patrón.

Estructura que debes crear

practica_app/ ├── index.php ← Front Controller │ ├── core/ │ ├── Router.php ← Lee URL, despacha │ └── View.php ← Carga vistas │ ├── controllers/ │ ├── HomeController.php │ └── LibroController.php │ ├── models/ │ └── Libro.php │ └── views/ ├── layout/ │ ├── header.php │ └── footer.php ├── home/ │ └── index.php └── libros/ ├── lista.php └── show.php
Controllers

Reciben la petición, llaman al Model y pasan los datos a la View. Cero HTML, cero SQL.

Model — Libro.php

Métodos: todos(), buscar($id), porGenero($g), generos(). Cero HTML.

Views

Solo HTML + variables que vienen del Controller. Cero SQL, cero lógica de negocio.

Las 8 tareas

  • 1
    core/Router.php Lee $_GET['c'] y $_GET['a'], carga el controller correcto y llama al método.
  • 2
    core/View.php Método estático render($vista, $datos): hace extract($datos) y carga header + vista + footer.
  • 3
    models/Libro.php Array estático con al menos 5 libros. Implementa todos(), buscar($id), porGenero($g) y generos().
  • 4
    controllers/HomeController.php Acción index(): renderiza home/index. Acción notFound(): respuesta 404.
  • 5
    controllers/LibroController.php Acción index(): todos los libros → vista libros/lista. Acción show(): un libro por id → vista libros/show. Acción genero(): libros filtrados.
  • 6
    views/layout/header.php y footer.php Layout compartido. El header usa $titulo. El footer incluye un enlace de regreso al catálogo.
  • 7
    views/libros/lista.php Muestra los libros en tarjetas con filtros por género. Los enlaces apuntan a ?c=libro&a=show&id=X.
  • 8
    views/libros/show.php Muestra todos los datos del libro: título, autor, año, precio, descripción. Botón para volver al catálogo.

Cómo probar

Arrancar el servidor
cd practica_app
php -S localhost:8080
URLs a verificar
  • localhost:8080/index.php — Página de inicio
  • ?c=libro&a=index — Lista todos los libros
  • ?c=libro&a=show&id=1 — Detalle del libro 1
  • ?c=libro&a=genero&g=Novela — Filtro por género
  • ?c=libro&a=show&id=999 — Debe redirigir a 404
El indicador de capa — La demo tiene una cajita en la esquina inferior derecha que muestra qué Controller, Model y View se están usando. Incluye algo similar en tu app para demostrar que comprendes el flujo.
